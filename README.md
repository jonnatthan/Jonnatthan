# 🧪 LabControl Pro

Sistema SaaS para gestión de laboratorios farmacéuticos con enfoque en cumplimiento normativo INVIMA, trazabilidad y control de calidad.

---

## 🚀 Descripción

LabControl Pro es una plataforma web diseñada para gestionar el ciclo completo de muestras en laboratorio:

* Solicitudes de análisis
* Recepción de muestras
* Análisis fisicoquímico y microbiológico
* Control de cambios (CC-AA-NNNN)
* Auditoría y trazabilidad completa

---

## 🧠 Problema que resuelve

En muchos laboratorios, el control de muestras y documentación se maneja en Excel o sistemas fragmentados.

Esto genera:

* Pérdida de trazabilidad
* Riesgo en auditorías INVIMA
* Errores humanos

LabControl centraliza todo en un solo sistema.

---

## ⚙️ Funcionalidades principales

* 🔐 Sistema de login por roles
* 📦 Gestión de muestras (FQI / MCI)
* 🔬 Módulo de análisis
* 📊 Dashboard con indicadores
* 📝 Control de cambios obligatorio
* 📜 Auditoría de acciones
* ⏰ Alertas de vencimiento

---

## 🛠️ Tecnologías

* HTML5
* CSS3
* JavaScript (Vanilla)
* LocalStorage (simulación de backend)
* SQL (estructura de base de datos)

---

## 🧱 Arquitectura

* Frontend SPA (Single Page Application)
* Gestión de estado en memoria + LocalStorage
* Modelo relacional en SQL (Users, Samples, Requests, Audit)
